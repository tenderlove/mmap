
 The Mmap class implement memory-mapped file objects 

 * Installation

    ruby extconf.rb
    make
    make install

 * Documentation

    make rd2
    make rdoc
    make ri

    See mmap.rd, mmap.html

 * Tests : if you have rubyunit

   make test
